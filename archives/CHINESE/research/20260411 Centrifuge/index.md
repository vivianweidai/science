## 概述

本实验对五种常见家用液体进行离心分离，观察其分离结果。同时练习了使用移液器、分析天平和pH计的基本实验室技术。

## 实验设置

| 类别 | 详情 |
|------|------|
| 离心机 | Thermo Scientific Sorvall RT3 Centrifuge |
| pH计 | VWR pHenomenal pH 1100 L，25 °C校准 |
| 涡旋混合器 | Scientific Industries Vortex-Genie 2 |
| 微型离心机 | DiaMed ID-Centrifuge |
| 磁力搅拌器 | Canlab Magnetic Stirrer |
| 天平 | Sartorius CP225D Analytical Balance |
| 移液器 | Gilson Pipetman Pipettes |
| 试管 | Sarstedt 50 mL锥形离心管，Falcon 50 mL试管 |

## 样品

<div class="hero-single"><img src="/research/projects/20260411 Centrifuge/PHOTOS/sample1.jpeg" alt="五个标记的Sarstedt试管：可乐、牛奶、酸奶、苹果醋、蜂蜜"></div>

五种日常液体被装入标记的50 mL离心管。每个样品先涡旋混匀，然后离心。离心前将VWR电极直接插入样品中测量一次pH值。

| 样品 | 说明 |
|------|------|
| 可乐 | 碳酸可乐软饮料 |
| 牛奶 | 全脂牛奶 |
| 酸奶 | 原味酸奶（液态） |
| 苹果醋 | 未过滤苹果醋 |
| 白醋 | 蒸馏白醋 |

## 结果

<div class="hero-single"><img src="/research/projects/20260411 Centrifuge/PHOTOS/sample8.jpeg" alt="四个标记的Sarstedt试管：1-牛奶 2-酸奶 3-苹果醋 4-牛奶+醋"></div>

四个样品被放入Thermo Scientific Sorvall RT3离心机中平衡转子（<a href="https://github.com/vivianweidai/science/blob/main/research/projects/20260411%20Centrifuge/PHOTOS/data16.jpeg">数据16</a>和<a href="https://github.com/vivianweidai/science/blob/main/research/projects/20260411%20Centrifuge/PHOTOS/data17.jpeg">数据17</a>）。离心后观察到明显分离：

- **管1 — 牛奶** — 分离为半透明乳清层和白色脂肪/酪蛋白沉淀（<a href="https://github.com/vivianweidai/science/blob/main/research/projects/20260411%20Centrifuge/PHOTOS/data24.jpeg">数据24</a>）
- **管2 — 酸奶** — 与牛奶类似的分离，但沉淀更致密（<a href="https://github.com/vivianweidai/science/blob/main/research/projects/20260411%20Centrifuge/PHOTOS/data19.jpeg">数据19</a>和<a href="https://github.com/vivianweidai/science/blob/main/research/projects/20260411%20Centrifuge/PHOTOS/data20.jpeg">数据20</a>）
- **管3 — 苹果醋** — 沉淀物（"醋母"）沉积在底部，上层为更澄清的醋（<a href="https://github.com/vivianweidai/science/blob/main/research/projects/20260411%20Centrifuge/PHOTOS/data18.jpeg">数据18</a>）
- **管4 — 牛奶+醋混合物** — 白色酪蛋白沉淀，上层为澄清乳清，酸使牛奶蛋白凝固，离心将其集中到底部（<a href="https://github.com/vivianweidai/science/blob/main/research/projects/20260411%20Centrifuge/PHOTOS/data21.jpeg">数据21</a>、<a href="https://github.com/vivianweidai/science/blob/main/research/projects/20260411%20Centrifuge/PHOTOS/data22.jpeg">数据22</a>和<a href="https://github.com/vivianweidai/science/blob/main/research/projects/20260411%20Centrifuge/PHOTOS/data23.jpeg">数据23</a>）

实验表明离心分离对乳液和悬浮液（牛奶、酸奶、未过滤醋）最有效，这些液体中不同密度的颗粒物理混合但未溶解。
