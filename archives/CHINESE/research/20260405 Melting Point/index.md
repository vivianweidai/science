## 概述

熔点测定是鉴定固体化合物和评估其纯度的基本技术。纯物质在特征温度下快速熔化，而杂质会扩大熔化范围并降低起始温度。本实验旨在使用自动熔点仪的毛细管法及数字图像处理技术测量咖啡因和阿司匹林的熔点。

**状态：** 本次实验期间OptiMelt仪器无法使用——触摸屏无响应，无法配置或启动运行。样品已制备但未收集任何测量数据。该项目暂搁，等待仪器维修或使用其他熔点仪器。

## 实验设置

| 类别 | 详情 |
|------|------|
| 仪器 | OptiMelt Automated Melting Point System |
| 方法 | 毛细管法，数字图像处理 |
| 毛细管 | Eisco Labs硼硅酸盐玻璃毛细熔管，4″长，0.05″外径 |
| 样品 | 咖啡因（CAF）、阿司匹林（ASP） |
| 重复 | 每种化合物3根毛细管 |

## 样品

<div class="hero-single"><img src="/research/projects/20260405 Melting Point/PHOTOS/20260405 Samples A.jpeg" alt="标记为CAF1–3和MP1–3的毛细管"></div>

样品被研磨成细粉并装入玻璃毛细管中（约2–3 mm填充高度）。每种化合物制备三个平行样（CAF 1–3, ASP 1–3）以便取平均值并评估重复性。毛细管标记后插入OptiMelt样品室。
