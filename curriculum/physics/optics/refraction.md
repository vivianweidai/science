# Refraction

| Label | Formula / Concept | Description |
|-------|-------------------|-------------|
| geometric | light as particles in rays | large scale vs wavelengths |
| geometric | reflection equals incidence | parallel at corner |
| refraction | vacuum to media | change direction |
| refraction | frequency constant | speed wavelength decrease |
| refraction | $n = \frac{c}{v}$ index | $n_{1}\sin\theta_{1} = n_{2}\sin\theta_{2}$ Snell |
| total internal reflection | slower to faster media | bend away from normal |
| total internal reflection | $\sin\theta_{c} = \frac{n_{2}}{n_{1}}$ | underwater view cone |
