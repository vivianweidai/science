# Congruence

| Label          | Formula                                                                   | Description    |
|----------------|---------------------------------------------------------------------------|----------------|
| congruence     | $a - b = mk$                                                              | definition     |
| congruence     | $a \equiv r_a \pmod{m}$                                                   | remainder      |
| addition       | $a + k \equiv r_a + k \pmod{m}$                                           | constant       |
| addition       | $a + b \equiv r_a + r_b \pmod{m}$                                         | sum            |
| multiplication | $ak \equiv r_a k \pmod{m}$                                                | scale          |
| multiplication | $ab \equiv r_a r_b \pmod{m}$                                              | product        |
| multiplication | $a^m \equiv r_a^m \pmod{m}$                                               | exponential    |
| division       | $ak \equiv bk \pmod{mk}$                                                  | common divisor |
| division       | $a \equiv b \pmod{\tfrac{m}{\gcd(k, m)}}$                                 | division       |
| factors        | $n \mid m,\ x \equiv y \pmod{m} \implies x \equiv y \pmod{n}$             | divisor        |
| factors        | $x \equiv y \pmod{m},\ x \equiv y \pmod{n} \implies x \equiv y \pmod{mn}$ | multiple       |
