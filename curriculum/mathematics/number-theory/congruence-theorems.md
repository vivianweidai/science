# Congruence Theorems

| Label   | Formula                                                            | Description        |
|---------|--------------------------------------------------------------------|--------------------|
| Chinese | $x \equiv a_i \pmod{m_i}$                                          | system congruences |
| Chinese | pair wise                                                          | relative primes    |
| Chinese | $x \equiv c \pmod{m_1 m_2 \ldots}$                                 | exists unique      |
| Chinese | specific solution                                                  | Euclidean          |
| Fermat  | $a^{p-1} \equiv 1 \pmod{p}$                                        | relative prime     |
| Fermat  | $a^{p} \equiv a \pmod{p}$                                          | not relative prime |
| Euler   | $a^{\varphi(n)} \equiv 1 \pmod{n}$                                 | relative prime     |
| Euler   | $a^{\operatorname{lcm}[\varphi(m), \varphi(n)]} \equiv 1 \pmod{mn}$ | multiplicative    |
