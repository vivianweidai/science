# Complex Numbers

| Label | Formula / Concept | Description |
|-------|-------------------|-------------|
| rectangular | $z = x + yi$ | complex plane |
| rectangular | real and imaginary | two components |
| polar | $z = r(\cos\theta + i\sin\theta)$ | substitution conversion |
| polar | factor and magnitude | reverse derive angle |
| exponential | $z = re^{i\theta}$ | definition conversion |
| exponential | simplifies rotations | derives identities |
