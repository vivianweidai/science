# Rules

| Label | Formula / Concept | Description |
|-------|-------------------|-------------|
| rules | $(fg)' = f'g + fg'$ | product |
| rules | $\left( \frac{f}{g} \right)' = \frac{f'g - fg'}{g^{2}}$ | quotient |
| rules | $(g \circ f)' = g'(f)f'$ | chain |
| rules | $(\sin x)' = \cos x$ | trigonometry |
| rules | $(e^{x})' = e^{x}$ | exponential |
| rules | $(\ln x)' = \frac{1}{x}$ | logarithmic |
| inverse | $\frac{d}{dx}f^{- 1}(x) = \frac{1}{f'(f^{- 1}(x))}$ | chain rule |
| inverse | $\frac{d}{dx}\sin^{- 1}x = \frac{1}{\sqrt{1 - x^{2}}}$ | inverse rule |
| inverse | $\frac{d}{dx}\tan^{- 1}x = \frac{1}{1 + x^{2}}$ | inverse trigonometric |
| implicit | $x^{2} + y^{2} = 1$ | chain rule |
| implicit | $y' = - \frac{x}{y}$ | group differentials |
| l\'Hopital | $\lim_{x \rightarrow a}\frac{f(x)}{g(x)} = \lim_{x \rightarrow a}\frac{f'(x)}{g'(x)}$ | zeroes or infinities |
