# Rules

| Label | Formula / Concept | Description |
|-------|-------------------|-------------|
| antiderivative | $\int x^{n}dx = \frac{x^{n + 1}}{n + 1} + C$ | power |
| antiderivative | $\int\frac{1}{\sqrt{1 - x^{2}}}dx = \sin^{- 1}x + C$ | trigonometric |
| antiderivative | $\int\frac{1}{x}dx = \ln\vert x\vert  + C$ | logarithmic construction |
| substitution | $\int g(f(x))f'(x)dx = \int g(u)du$ | u-substitution |
| substitution | $\int udv = uv - \int vdu$ | by parts |
| substitution | $\frac{5 - x}{(x - 2)(3 - x)} = \frac{A}{x - 2} + \frac{B}{3 - x}$ | partial fractions |
| parametric | $\int_{a}^{b}\sqrt{1 + (f'(x))^{2}}dx$ | arc length |
| polar | $A = \int\frac{1}{2}r^{2}d\theta$ | small angle |
| improper | $\int_{a}^{\infty}f(x)dx = \lim_{b \rightarrow \infty}\int_{a}^{b}f(x)dx$ | convergence if limit exists |
