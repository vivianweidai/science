# Mitotic Phase

| Label | Formula / Concept | Description |
|-------|-------------------|-------------|
| prophase | chromatin to chromosomes | sister chromatids tied centromere |
| prophase | centrioles at centrosomes | make microtubules |
| prometaphase | nuclear membrane disintegrates | centrosomes move to cell sides |
| prometaphase | kinetochore spindles | grab chromosomes |
| metaphase | chromosomes aligned | on metaphase plate |
| anaphase | chromosomes pulled apart | nonkinetochore elongate cell |
| telophase | nuclear membrane forms | chromosomes unravel chromatin |
| cytokinesis | cells split at cleavage furrow | plant cells new cell wall |
