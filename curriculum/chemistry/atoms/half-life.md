# Half Life

| Label | Formula / Concept | Description |
|-------|-------------------|-------------|
| half life | time to decay | to half original amount |
| half life | $\frac{dN}{dt} = - kN$ | first order differential equation |
| half life | $\ln N = - kt + \ln N_{0}$ | separation of variables |
| half life | $\ln\left( \frac{N}{N_{0}} \right) = - kt$ | rearrange |
| half life | $t_{\frac{1}{2}} = \frac{\ln 2}{k} = \frac{0.693}{k}$ | plug in half amount |
