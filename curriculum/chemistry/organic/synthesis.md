# Synthesis

| Label | Formula / Concept | Description |
|-------|-------------------|-------------|
| alkene | create $Br_{2}hv/NaOEt$ | radical bromination elimination |
| alkene | cleave $O_{3}/DMS$ | ozonolysis to $C = O$ |
| halogenation | $Br_{2}$ adds $BrBr$ | via bromonium bridge |
| halogenation | none | anti |
| halohydrin | $Br_{2}/H_{2}O$ adds $BrOH$ | via bromonium bridge |
| halohydrin | Markovnikov for alcohol | anti |
| dihydroxylation | add $OHOH$ | via epoxide |
| dihydroxylation | $RCO_{3}H\text{orMCPBA}/H_{3}O^{+}$ | none and anti |
| dihydroxylation | $OsO_{4}/H_{2}O_{2}\text{or}KMnO_{4}/NaOH$ | none and syn |
