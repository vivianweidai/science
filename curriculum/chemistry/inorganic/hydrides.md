# Hydrides

| Label | Formula / Concept | Description |
|-------|-------------------|-------------|
| hydrogen | strong $H - H$ not reactive | requires catalyst to react |
| hydrogen | borderline electronegativity | forms hydrides with all |
| covalent | hydrogen as metal $HCl$ | formal oxidation state $+ 1$ |
| covalent | $H$ high ionization energy | extreme charge density |
| covalent | molecular units $H_{2}ONH_{3}$ | with all nonmetals |
| covalent | polymeric $e^{-}$ deficient $BeH_{2}$ | with beryllium and 3A |
| ionic | hydrogen as nonmetal $LiH$ | formal oxidation state $- 1$ |
| ionic | $H^{-}$ low electron affinity | strong electron repulsion |
| ionic | salt compounds | with alkali and alkaline |
| ionic | white gray solids | drying reducing bases |
| metallic | nonstoichiometric $PdH_{0 - 0.7}$ | with transition metals |
| metallic | interstitial crystals | store purify hydrogen |
