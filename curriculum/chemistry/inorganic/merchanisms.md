# Merchanisms

| Label | Formula / Concept | Description |
|-------|-------------------|-------------|
| mechanisms | coordination reactions | changes to metal or ligands |
| mechanisms | most common is substitution | octohedral or square planar |
| octohedral | first row metals labile | $Cr^{3 +}Co^{3 +}$ other rows inert |
| octohedral | dissociation not association | evidenced by reaction rates |
| octohedral | larger charge size less steric | stronger $M - L$ bonds |
| octohedral | intermediate five coordinate | negative $\Delta CFSE$ more $E_{A}$ |
| redox | electron transfer | from one metal to other |
| redox | outer sphere coordinations intact | $M - L$ distance difference |
| redox | inner sphere bridging complex | larger halide more polarizable |
| square planar | association not dissociation | readily attack above below |
| square planar | kinetic trans effect | more polarizable trans directing |
| square planar | $CN^{-} > I^{-} > Br^{-} > Cl^{-} > NH_{3} > H_{2}O$ |  |
