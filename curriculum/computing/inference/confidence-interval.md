# Confidence Interval

| Label | Formula / Concept | Description |
|-------|-------------------|-------------|
| confidence interval | likelihood that interval | contains population parameter |
| confidence interval | accept cause and effect claim | if value is inside the interval |
| confidence interval | $\text{point estimate} \pm \text{margin of error}$ |  |
| confidence interval | $\text{margin of error} = \text{(critical value) (standard error)}$ |  |
| proportion | $\widehat{p} \pm z^{*}\sqrt{\frac{\widehat{p}(1 - \widehat{p})}{n}}$ |  |
| proportion | $({\widehat{p}}_{1} - {\widehat{p}}_{2}) \pm z^{*}\sqrt{\frac{{\widehat{p}}_{1}(1 - {\widehat{p}}_{1})}{n_{1}} + \frac{{\widehat{p}}_{2}(1 - {\widehat{p}}_{2})}{n_{2}}}$ |  |
| mean | $\overline{x} \pm t^{*}\frac{s_{x}}{\sqrt{n}}$ | $df = n - 1$ |
| mean | $({\overline{x}}_{1} - {\overline{x}}_{2}) \pm t^{*}\sqrt{\frac{s_{1}^{2}}{n_{1}} + \frac{s_{2}^{2}}{n_{2}}}$ |  |
| slope | $b \pm t^{*}\frac{s}{s_{x}\sqrt{n - 1}}$ | $df = n - 2$ |
