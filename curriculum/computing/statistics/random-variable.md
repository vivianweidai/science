# Random Variable

| Label | Formula / Concept | Description |
|-------|-------------------|-------------|
| discrete | assigns numerical values | to random process outcomes |
| discrete | probability distribution | possible values and probabilities |
| discrete | long run outcomes | not individual probabilities |
| discrete | $\mu_{X} = E(X) = \sum x_{i}P(x_{i})$ |  |
| discrete | $\sigma_{X} = \sqrt{\sum(x_{i} - \mu_{X})^{2}P(x_{i})}$ |  |
| combinations | combine random variables | linearity of expected value |
| combinations | addition $X + k \rightarrow \mu + k,\sigma$ | scaling $kX \rightarrow k\mu,k\sigma$ |
| combinations | $aX + bY \rightarrow a\mu_{X} + b\mu_{Y},\sqrt{a^{2}\sigma_{X}^{2} + b^{2}\sigma_{Y}^{2}}$ |  |
