# Repeated

| Label | Formula / Concept | Description |
|-------|-------------------|-------------|
| binomial | fixed number trials | count total successes |
| binomial | $P(X = x) = \binom{n}{x}p^{x}(1 - p)^{n - x}$ |  |
| binomial | $\mu_{x} = np$ | $\sigma_{x} = \sqrt{np(1 - p)}$ |
| geometric | expected number of trials | until first success |
| geometric | $P(X = x) = (1 - p)^{x - 1}p$ |  |
| geometric | $\mu_{x} = \frac{1}{p}$ | $\sigma_{x} = \frac{\sqrt{1 - p}}{p}$ |
