# Compound

| Label | Formula / Concept | Description |
|-------|-------------------|-------------|
| sum | $a + b$ | sum of independent variables |
| sum | $\left( \Delta(a + b) \right)^{2} = (\Delta a)^{2} + (\Delta b)^{2}$ | absolute add in quadrature |
| sum | $\Delta(a + b) < \Delta a + \Delta b$ | weighted average lowest total |
| average | $\overline{x} = \frac{x_{1} + x_{2} + \ldots x_{n}}{n}$ | with same independent errors |
| average | $\Delta\overline{x} = \frac{1}{n} \cdot \left( \sqrt{(\Delta x_{1})^{2} + \ldots} \right)$ | by scaling and sum |
| average | $\Delta\overline{x} = \frac{1}{\sqrt{n}} \cdot \Delta x$ | reduced with sample squared |
| product | $ab = a \cdot b$ | product of independent variables |
| product | $\left( \frac{\Delta ab}{ab} \right)^{2} = \left( \frac{\Delta a}{a} \right)^{2} + \left( \frac{\Delta b}{b} \right)^{2}$ | relatives add in quadrature |
