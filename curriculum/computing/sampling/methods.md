# Methods

| Label | Formula / Concept | Description |
|-------|-------------------|-------------|
| SRS | simple random sample | equal chance being chosen |
| SRS | sample with or without | replacement of chosen |
| stratified | strata groups of individuals | thought to share characteristics |
| stratified | combine SRS from each strata | well chosen more precise |
| cluster | cluster groups of individuals | who are near each other |
| cluster | more practical to use | for large populations |
| systematic | ordered population | select every $k$th individual |
| systematic | when SRS impossible | such as polling station |
