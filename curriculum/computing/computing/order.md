# Order

| Label | Formula / Concept | Description |
|-------|-------------------|-------------|
| binary search | $n \leq 10^{10}$ | $O(\log n)$ |
| simple search count sort | $n \leq 10^{7}$ | $O(n)$ |
| merge sort | $n \leq 10^{7}$ | $O(n\log n)$ |
| selection sort | $n \leq 10^{4}$ | $O(n^{2})$ |
| triple for loop | $n \leq 550$ | $O(n^{3})$ |
| subsets | $n \leq 25$ | $O(2^{n})$ |
| travelling salesman | $n \leq 10$ | $O(n!)$ |
