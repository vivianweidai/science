# Magnitude

| Label | Formula / Concept | Description |
|-------|-------------------|-------------|
| apparent | luminosity distance | logarithmic flux |
| apparent | $m_{2} - m_{1} = - 2.5\log\left( \frac{F_{2}}{F_{1}} \right)$ | difference $5$ factor $100$ |
| apparent | $m = C - 2.5\log F$ | Vega zero |
| apparent | $m = C - 2.5\log L + 2.5\log(4\pi) + 5\log d$ |  |
| absolute | luminosity only | distance $10$ parsecs |
| absolute | $M = C - 2.5\log L + 2.5\log(4\pi) + 5$ |  |
| distance modulus | apparent vs absolute | distance vs $10$ parsecs |
| distance modulus | $m - M = 5\log d - 5$ | logarithmic distance |
